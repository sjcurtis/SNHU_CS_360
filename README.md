# SNHU CS 360

The goal of the app was to allow users to manage their inventory. Users could add inventory and update the quantities. Users could also delete items from their inventory.

A couple of screens were designed for the app. There was a login screen that allowed users to register and login. Another screen to display a list of items that were being tracked. One last screen was added to allow users to add items.

My goal was to keep the design simple for the class. It also made it easier to implement. A simple design makes it easy to interact with. Testing was a manual process. It did reveal bugs as I made changes and worked out the final version.

I think the biggest challenge was the actual implementation especially around the recycler view. There was a lot to learn but it did simplify implementation.

The login screen is a good demonstration of my skills. It takes input from multiple fields and adds them to a database. I then queried the database to verify a login is valid.
